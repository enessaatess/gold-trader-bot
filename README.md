# Gold Trader Bot

AI-powered virtual gold trading bot using Groq API and GitHub Actions.

## Setup

1. Create a public GitHub repository
2. Add your API keys to GitHub Secrets:
   - `GOLDAPI_KEY`
   - `OPENEXCHANGE_KEY`
   - `NEWS_API_KEY`
   - `GROQ_API_KEY`
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `INITIAL_GOLD_GRAMS`
   - `INITIAL_CASH_TRY`

3. Run the workflow from GitHub Actions
